About This Project

This real-time weather monitoring system provides a user-friendly dashboard to track weather conditions in multiple cities. Users can:

  • Monitor temperature, feels-like temperature, and weather conditions.
  • View visualizations for better insights.
  • Receive alerts for extreme weather conditions.

The system fetches data using the OpenWeatherMap API and stores it in a database for processing and visualization.
